## [1.0.145] - 2026-01-28

### Features

- **AI-Powered Reminder Notifications**: Dynamic, natural reminder messages generated by AI
  - New `reminder-message-generator.ts` service using GPT-4.1-NANO for message generation
  - Language detection from original reminder text - responds in same language (Hebrew, Arabic, English, etc.)
  - Graceful fallback to simple bold formatting if AI generation fails
  - Works across all channels: WhatsApp, Telegram, Slack, and Email

- **Multi-Platform Reminder Improvements**: Enhanced reminder system across all bot platforms
  - Bulk reminder creation without unnecessary confirmations
  - Improved time parsing with friendly error messages
  - Past-time detection with helpful feedback ("That time has already passed X minutes ago...")

### Bug Fixes

- **Conversation Context Loss ("Goldfish Effect")**: Fixed bot forgetting context after confirmations
  - Increased `MAX_CONTEXT_LENGTH` from 1500 to 6000 characters
  - Increased `MAX_MESSAGES_BEFORE_SUMMARIZE` from 6 to 10 messages
  - Added `MIN_MESSAGES_TO_KEEP = 4` to preserve recent conversation
  - File: `be/whatsapp-bot/utils/conversation-history.ts`

- **Intent Hallucination**: Fixed "remind me in 5 mins" being interpreted as "save my name as Jimmy"
  - Added explicit exclusions for reminder requests in naming intent detection
  - Changed naming examples from "Call me X" to "My name is X" to avoid confusion
  - File: `be/ai-agents/agents-instructions.ts` (lines 552-595)

- **DPO Optimizer Confirmation Loop**: Fixed bot asking "Are you sure?" before creating reminders
  - Added `<reminder_exception>` section to DPO optimizer excluding reminders from safety checks
  - Added "REMINDERS (any quantity)" to `<when_not_to_optimize>` list
  - File: `be/ai-agents/dpo/optimizer-agent.ts`

- **Agent Instructions Confirmation Override**: Prevented AI from injecting confirmations for reminders
  - Added "CRITICAL - NO CONFIRMATION NEEDED (EVEN FOR MULTIPLE REMINDERS)" section
  - Added explicit examples for bulk reminder handling
  - File: `be/ai-agents/agents-instructions.ts`

- **Orphaned Yes/No Handling**: Fixed user saying "yes" with no pending confirmation showing menu
  - Added fallback response: "I don't have anything pending to confirm. What would you like to do?"
  - File: `be/whatsapp-bot/handlers/message-handler.ts`

- **Processing Indicator**: Removed hardcoded processing message, letting AI respond directly
  - Removed `showProcessingIndicator()` calls from message handler
  - File: `be/whatsapp-bot/handlers/message-handler.ts`

- **Reminder Error Messages**: Improved error messages for reminder creation failures
  - "Invalid scheduled time format" → "Could not parse the time '...'. Please provide a valid time..."
  - "Scheduled time must be in the future" → "That time has already passed (X minutes ago)..."
  - "Failed to create reminder" → "Could not save the reminder. Please try again in a moment."
  - File: `be/domains/reminders/tools/reminder-tools.ts`

### Technical

- **9 commits** for reminder system fixes and improvements
- **New File**: `be/domains/reminders/services/reminder-message-generator.ts` - AI-powered message generation
- **Updated Files**:
  - `be/whatsapp-bot/utils/conversation-history.ts` - Context retention improvements
  - `be/whatsapp-bot/handlers/message-handler.ts` - Processing flow fixes
  - `be/ai-agents/agents-instructions.ts` - Intent disambiguation and confirmation prevention
  - `be/ai-agents/dpo/optimizer-agent.ts` - Reminder exception handling
  - `be/domains/reminders/services/reminder-dispatcher.ts` - AI message integration
  - `be/domains/reminders/tools/reminder-tools.ts` - User-friendly error messages
  - `fe/app/(marketing)/changelog/page.tsx` - Frontend changelog update

### Cross-Platform Impact

All fixes apply to shared code, benefiting:
- WhatsApp Bot (primary testing platform)
- Telegram Bot (same DPO optimizer and agent instructions)
- Slack Bot (same DPO optimizer and agent instructions)
- Web Chat (same agent instructions)
